# Exploratory Data Analysis (EDA)

This project performs exploratory data analysis on a retail dataset to understand sales patterns, product performance, and inventory behavior.

## Overview

* Dataset: Retail sales data
* Techniques: Data cleaning, aggregation, visualization
* Tools: Python (Pandas, Matplotlib)

## Analysis Performed

* Data inspection and summary statistics
* Missing value check
* Product-wise sales analysis
* Category-wise distribution
* Time-based sales trend
* Inventory analysis

## Project Structure

* data/ : dataset
* notebooks/ : Jupyter notebook
* scripts/ : Python scripts
* images/ : visualization outputs

## Results

* Identified top-performing products
* Detected low inventory items
* Observed sales trends over time
